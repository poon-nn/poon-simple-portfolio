# Simple Portfolio - Nutthaphat Poon

This is a personal portfolio website for Nutthaphat Poon, a Product Manager. It features a modern, dark-themed design and is built with HTML, CSS, and JavaScript.

## Features
- **Modern Dark Theme:** Default dark mode with an option to toggle to a light theme.
- **Responsive Design:** Adapts to different screen sizes.
- **JetBrains Mono Font:** Clean, technical typography.

## Deployment
This project is deployed using GitHub Pages.

## Tech Stack
- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
