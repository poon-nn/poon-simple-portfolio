# Portfolio Website Plan - Nutthaphat Poon (Product Manager)

## Objective
Create a professional, modern, dark-themed personal portfolio website for Nutthaphat Poon (Product Manager) hosted on GitHub Pages.

## Specs & Design
- **Theme:** Modern Dark Theme.
- **Typography:** JetBrains Mono (monospaced for a technical, clean aesthetic).
- **Structure:**
    1.  **Hero Section:** "Nutthaphat Poon - Product Manager" headline.
    2.  **Projects Section:** Showcase 3 key projects (with descriptions, tech stack, and links).
    3.  **Contact Section:** Links to professional platforms (LinkedIn, Email, GitHub).
- **Hosting:** GitHub Pages.

## Implementation Steps

### Phase 1: Setup & Environment
1.  Initialize a new GitHub repository named `portfolio`.
2.  Create local folder structure:
    - `index.html`
    - `/css/style.css`
    - `/js/main.js` (for minimal interactivity if needed)
    - `/assets/` (for images/icons)

### Phase 2: Core Development
1.  **HTML Structure:** Build semantic HTML for Hero, Projects, and Contact sections.
2.  **Styling (CSS):**
    - Implement CSS Variables for a modern dark color palette.
    - Import JetBrains Mono via Google Fonts.
    - Ensure responsive design.
3.  **Content Population:**
    - Insert Name & Title.
    - Placeholder cards for 3 projects.
    - Contact links.

### Phase 3: Deployment
1.  Push code to GitHub repository.
2.  Navigate to **Settings** > **Pages** in the repo.
3.  Configure **Source** to "Deploy from a branch" (branch: `main`, folder: `/`).
4.  Verify live site at `https://[username].github.io/portfolio/`.

## Verification & Testing
- Validate mobile responsiveness on various viewport widths.
- Ensure all links (Contact, Project URLs) are functional.
- Check font rendering and color contrast consistency.
